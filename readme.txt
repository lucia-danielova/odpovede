⚙️ Inštalácia a spustenie
Súbory skopírujte na lokálny PHP server (napr. XAMPP, MAMP, alebo vlastný Apache/Nginx).

Uistite sa, že server má povolené file_get_contents pre externé URL (potrebné pre ARES API).

Otvorte index.html vo vašom prehliadači.

💡 Kľúčové vlastnosti riešenia
Separation of Concerns: Kód je striktne rozdelený na štruktúru (HTML), dizajn (CSS) a logiku (JS).

User Experience: Implementovaný Dark Mode, Toast notifikácie a vizuálne indikátory načítavania (Loader).

Bezpečnosť a Validácia: Vstupné IČO je sanitizované na backendovej aj frontendovej úrovni.

Export dát: Možnosť stiahnuť výsledky z ARES alebo duplicitné záznamy do formátu CSV alebo kopírovať priamo do tabuľkového procesora (Google Sheets/Excel).
